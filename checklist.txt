FT_PRINTF - CHECKLIST


ATTENTION : Ordre des priorités ET CONCEPTION

Le Piège : Si vous codez la partie obligatoire de manière simple, 
vous devrez tout recommencer pour ajouter les bonus (flags, largeur). 
C'est vrai.

La Solution : Votre code doit, dès le début, être "extensible". 
Il doit être capable de "parser" (analyser) les flags (-0. # +) et la largeur, 
avant de savoir si c'est un %d ou un %s.

L'ordre de travail est :

Concevoir un système qui peut gérer les bonus.

Coder la partie obligatoire en utilisant ce système (elle doit être parfaite).





va_list = type de la variable qui va contenir la liste d'arguments. 
quon utilisera dans va_start	va_list ma_boite_a_arguments;


va_start = ouvre va_list		va_start(ma_boite_a_arguments, dernier_argument_fixe);
fonction qui ouvre la boite 

va_arg = sert a lire argument un par un, et pointe vers le prochain arguments.
int prochain_int = va_arg(ma_boite_a_arguments, int);

va_end = liberer tout mes copain





Phase 1 : La Base (Conception Extensible)

[] Comprendre les fonctions variadiques (va_start, va_arg, va_end).

[] Planifier la structure de parsing (une fonction qui lit les flags -0. 
et la largeur, même si vous ne les utilisez pas encore).

[] Créer la structure principale (parser la chaîne, trouver le '%').

[] Gérer le cas %%.




Phase 2 : Conversions Obligatoires

[] Gérer %c (caractère).

[] Gérer %s (chaîne).

[] Gérer %p (pointeur en hexadécimal).

[] Gérer %d (décimal).

[] Gérer %i (entier).

[] Gérer %u (décimal non signé).

[] Gérer %x (héxadécimal minuscule).

[] Gérer %X (héxadécimal majuscule).




Phase 3 : Tests et Finitions (Obligatoire)

[] Tester chaque conversion (cas normaux, cas limites, NULL, 0, etc.).

[] S'assurer que le comportement est identique au vrai printf().

[] Vérifier 0 fuite mémoire (malloc/free si utilisés).

[] Vérifier le Makefile (règles : all, clean, fclean, re, NAME).

[] Vérifier la norme (Norminette) sur TOUS les fichiers.

[] La partie obligatoire doit être PARFAITE.




Phase 4 : Bonus (Seulement si Phase 3 est parfaite)

[] Implémenter la logique pour les flags : -, 0, ..

[] Implémenter la logique pour la largeur de champ minimale.

[] Implémenter la logique pour les flags : #, +, (espace).

[] Retester TOUT (obligatoire + bonus).
